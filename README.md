# Mini-Project---1
DS_Global Seismic Trends
This project analyzes global earthquake data collected from the USGS Earthquake API over the past five years.

The data was extracted in JSON format using Python requests and converted into a Pandas DataFrame.

Data cleaning techniques such as null-value handling, duplicate removal, and feature engineering were performed.

Additional columns like country and continent were created for regional earthquake analysis.

The cleaned dataset was stored in a MySQL database and analyzed using advanced SQL queries.

Streamlit was used to build an interactive dashboard for visualizing earthquake trends, magnitude, depth, and tsunami alerts.

The project helps identify seismic patterns, high-risk regions, and earthquake trends for better data-driven insights.
